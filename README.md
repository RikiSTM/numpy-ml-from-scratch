# NumPy ML from Scratch

A collection of classical Machine Learning algorithms implemented from the ground up using pure NumPy. This project focuses on bridging the gap between mathematical theory and production-ready code through a rigorous software engineering lens.

## 🚀 Goals
- **Zero High-Level Abstractions:** Building models without relying on Scikit-Learn or similar frameworks.
- **Vectorization First:** Prioritizing efficient matrix-based operations over native Python loops to maximize performance.
- **Algorithmic Rigor:** Including Big O Notation complexity analysis for both time and space for every implementation.
- **Engineering Excellence:** Implementing MLOps best practices, including deterministic dependency management and unit testing.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Math Engine:** NumPy
- **Dependency Management:** [Poetry](https://python-poetry.org/)
- **Testing:** Pytest
- **Version Control:** Git (following Conventional Commits)

## 📂 Project Structure
- `core/`: Core algorithm logic (Linear Models, Decision Trees, Optimizers, etc.).
- `experiments/`: Practical case studies, performance benchmarks, and visualization scripts.
- `tests/`: Unit tests to validate mathematical correctness and matrix dimensions.
